Training

We offer time tested, hands-on training on emerging web technologies for developer teams

Course Title

Develop large scale Web application using ReactJS

Course Outline

You will learn how to develop large scale web application using ReactJS. During the course, we will be developing and unit testing a full-fledged React based web application. Also we will deploy the app onto cloud.

Course Duration

2 days (16 hours)

Level

Intermediate

Pre-requisites

  • Basic JavaScript knowledge is required
  • Knowledge of client MVC frameworks will be an added advantage

What you will learn

  • Understand React essentials to build large scale web applications
  • Cloud based database
  • Handle Real-time notifications
  • Unit testing React applications
  • Make your app mobile ready
  • Workflow in developing the React applications

Topics

  • Web applications - evolution
  • Introduction to React
  • Components and configuration
  • Component internal state and lifecycle
  • Communicating among components
  • Container components
  • Real-time events
  • Forms
  • Routing
  • Testing
  • Make your app mobile ready
  • Application data management
